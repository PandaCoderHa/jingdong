返利，泛指供应商针对京东的采购、销售、回款等业务提供的各类费用支持。

n   返利类型

返利类型

说明

规模返利

基于一段时间采购/销售/回款业务量为计算基准的返利

定额费用

供应商因商品促销、市场推广、新品引进等原因，直接确认一笔固定费用

毛利保护

与供应商约定月度毛利保护点位，若实际毛利率低于约定毛利点位，向供应商收取差额

促销返利

京东或与供应商一起开展的各种促销让利活动，供应商承担的让利部分

价格保护

由于市场变化，产品采购价调低，供应商对京东库存商品或期间采购商品进行补差

n   返利结算流程

1.      录入―主要应用于合同外返利-人工手动录入金额或规则

2.      调整―主要分为常规调整、减免调整

3.      下传―供应商确认返利可以结算，返利金额从返利系统分至结算系统

4.      回收―返利通过账扣或电汇的结算

n   返利注意事项

1.      合同内返利注意事项： 合同有效期为1年（多数情况下），到期后原合同自动延期90天，返利需手动延期90天；当供应商返利政策更新后要及时签署合同补充协议，完善返利内容。

2.      合同外返利管理：合同外返利政策须以供应商函件确认为准，函件支持书面（电子）公函（海外直采供应商常用）、授权邮箱发出的邮件两种。需手动录入至返利系统。

n   返利录入原则

1.       及时性：返利政策一经供应商确认，采销员两个工作日内将返利确认函件提报给返利录入岗，并做好返利政策备案。录入岗应在两个工作日内按照返利录入合规要求录入系统并提交审核，返利审核岗应在一个工作日内审核完毕。

2.       清晰、严谨性：返利政策约定必须清晰、明确、严谨、细致，避免模糊、异议条款，确保返利及时、准确回收。

3.       提高合同内返利占比原则,在与供应商合作过程中，采销人员应逐步提高合同内返利占比，明确公司预期收益，提高返利管理效率，降低后续谈判成本。

n   返利失效

1.       返利失效的原因：协议临时终止、协议变更、协议录入错误等情况都会造成返利失效。

2.       返利失效是在财务已审核，但是未生成返利单的情况下，才能进行的操作。

（协议变更和协议录入错误，需要重新录入正确的返利，才能使错误的返利失效。）

由于不同事业部的返利管理方式不同，如需了解更多返利相关内容，请与本部门同事或返利岗沟通。
